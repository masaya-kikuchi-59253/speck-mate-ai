# SpecMate AI

SpecMate AI は、電子部品の納入仕様書を AI で自動解析し、チェックリストとの照合を支援するツールです。

## 概要（モック版）

本アプリケーションは、**機能検証用のモックアップ**です。実際の AI 解析エンジンとは接続されておらず、あらかじめ定義されたデータに基づいて動作をシミュレートします。

- **ファイル処理機能**:
    - **アップロード**: PDF や Word 文書をサーバーにアップロードし、ブラウザで閲覧できます。アップロードされたファイルは `backend/uploads/` に保存されます。
    - **AI 解析（模擬）**: 各チェック項目に対して「AI が解析した」という想定の固定的な提案内容を表示します。実際のファイル内容の解析は行われません。
- **UI 機能**:
    - **部品サマリ・チェックリスト**: 1608 サイズのセラミックコンデンサを想定したサンプル情報を閲覧・編集できます。
    - **ヒューマン・イン・ザ・ループ**: AI の（模擬）提案を人間が確認し、判定を変更するプロセスを体験可能です。
    - **エクスポート**: 確認した結果を Excel ファイルとしてダウンロードできます。

### セキュリティに関する注意事項

本ツールは**プライベートネットワーク内での使用を想定**しています。以下の点にご注意ください：

- CORS 設定がすべてのオリジンを許可しているため、パブリックネットワークでの使用は推奨されません。
- アップロードされたファイルは認証なしでアクセス可能です。
- **社内ネットワークやローカルネットワーク内でのみ使用してください。**
- 機密情報を含むファイルを扱う場合は、適切なネットワーク隔離を行ってください。

## 前提条件

本ツールを動作させるには、実行環境（サーバー）に **Node.js** がインストールされている必要があります。

- **Node.js**: v18.0.0 以上
- **Node.js インストーラー**: [https://nodejs.org/](https://nodejs.org/) から推奨版（LTS）をダウンロードしてインストールしてください。

## 起動方法

### 1. サーバーで起動

Node.js のインストール完了後、リポジトリルートにある `start.bat` を実行することで、フロントエンドとバックエンドの両方が起動します。

```batch
start.bat
```

### 2. アクセス方法

起動後、以下の URL にアクセスしてください：

**サーバー自身からのアクセス:**
- **画面（Frontend）**: [http://localhost:5173](http://localhost:5173)

**ネットワーク上の他の PC からのアクセス:**
- **画面（Frontend）**: `http://<サーバーのIPアドレス>:5173`
- 例: `http://192.168.1.100:5173`

※ファイアウォールでポート 5173 (Frontend) と 3001 (Backend) の通信を許可してください。
